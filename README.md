# Busly 🚌

## Overview
Busly is a smart Android application developed using Kotlin to help users track buses, view routes, and access travel information.

## Features
- 🚌 Bus route information
- 📍 Bus tracking
- 🔎 Search buses
- 📱 Simple and user-friendly interface

## Technologies Used
- Kotlin
- Android Studio
- Gradle

## Future Enhancements
- Live GPS tracking
- Real-time bus arrival time
- Push notifications
- Online ticket booking
